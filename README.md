# gobang
带禁手的五子棋（中国科学院大学 杨力祥老师的公选课《C++程序设计》大作业）<br />
时间仓促，代码略显潦草，仅供参考！


## 类说明

| 类            | 说明                                        |
| ------------ | ----------------------------------------- |
| Game         | 游戏类，控制游戏的流程，引导玩家操作，例如：选择游戏模式、选择先手玩家，开始游戏等 |
| Checkerboard | 棋盘类，保存棋盘数据，提供下棋接口和展示棋盘的接口                 |
| ChessPiece   | 棋子类                                       |
| Judge        | 裁判类，判定五连和禁手                               |
| Player       | 真人玩家类                                     |
| Machine      | 电脑玩家类                                     |
